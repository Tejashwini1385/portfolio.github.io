Lost and Found Portal

A full-stack web application to help users report and find lost and found items with login and register functionality.

Frontend: Built using React.js for dynamic UI components, styled with HTML and CSS.

Backend: Developed using Node.js and Express to handle API logic.

Database: MySQL hosted on a local WAMP server to store user, item, and feedback data.

Deployment: Hosted on Google Cloud Platform (GCP) as an IaaS service using an external IP for access.

Users can register or log in, and after authentication, they can post a lost or found item by providing a name and uploading an image. Items are categorized as lost or found for easy navigation. Users can also submit feedback. Admins have a separate login to manage item data and have exclusive access to delete any listed item.

Ensemble and Attention Mechanisms for Fine-Grained Image Recognition

A deep learning-based framework for high-precision fine-grained image classification using ensemble and attention mechanisms.

Overview: Fine-Grained Image Classification (FGIC) refers to the task of separating visually similar categories with slight variations. This is a challenging task due to high intra-class variability and inter-class similarities.

Languages & Tools: Python, PyTorch, Google Colab

Implementation: This research proposes a novel ensemble attention-based transfer learning framework combining EfficientNet B0, DenseNet121, and ResNet152, each integrated with Convolutional Block Attention Module (CBAM). The architecture enhances feature extraction and focuses on discriminative regions, effectively tackling challenges like occlusion and scale variation.

Performance: The model achieved 94.25% accuracy on the CUB-200-2011 dataset, outperforming state-of-the-art models such as Vision Transformers (ViT) and API-Net, while maintaining a balance between accuracy and computational efficiency.

Research Recognition: This work has been accepted for publication in the Springer Lecture Notes series as part of the CRM 2025 Conference.

Techniques Used: Transfer Learning, Ensemble Learning, CBAM, Attention Mechanisms, K-Fold Cross-Validation

High-Resolution Image Reconstruction

A deep learning framework for reconstructing high-quality images from low-resolution inputs using advanced GAN and attention mechanisms.

Languages & Tools: Python, PyTorch, Google Colab

Implementation: High-resolution image reconstruction is achieved by creating highly coherent and detailed images that persist in their fine features alongside the broader contextual information. This was further explored by StyleSwin, by incorporating the modified multi-grain attention mechanism inside the generative adversarial network (GAN). In doing so, the model utilizes the Swin transformer in the style-based architecture to yield the optimal balance between computational efficiency and modeling capacity. Our attention mechanism modifications allow StyleSwin to capture both the fine detail and broader context. To encourage spatial coherence in high-resolution outputs, StyleSwin uses sinusoidal positional encoding and features a wavelet discriminator to mitigate blocking artifacts. Comprehensive experimentation on the FFHQ dataset shows that StyleSwin substantially outperforms earlier transformer-based GANs, state-of-the-art for high-resolution image synthesis and reconstruction.
